Frequently Asked Questions

How do I transfer my license to a new computer?
If you are using a Software Key, you must deactivate the key on your old computer using the Fine License Manager before activating it on the new machine. If you are using a physical USB hardware key, simply plug the USB drive into the new computer and install the latest drivers.
Can I open newer files in an older version of the software?
No, files created in newer editions (e.g., Edition 2026) cannot be opened in older editions (e.g., Edition 2025) due to structural changes in how mathematical models are saved. However, newer editions can always open files created in any older version.
How frequently are engineering standards updated?
We release a major Edition upgrade once a year (typically in November) and a Spring Update in May. Both updates include the latest implementations of Eurocodes, AASHTO, and regional annexes. Active Fine Maintenance subscribers receive these updates automatically.
What is included in the Demo version?
The Demo version includes all features of the full software without time limitations. However, saving data files is disabled, printing calculation reports is watermarked, and exporting data (DXF, IFC) is restricted. It is strictly designed for evaluation purposes.
