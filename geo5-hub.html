1. Centralized Geological Model

Create a single 3D subsoil model using the Stratigraphy program. Define boreholes, CPTs, and soil layers once. This acts as the single source of truth for your entire project.

2. 1D Cross-Section Extraction

Instantly extract a 2D cross-section from the 3D model. The profile, including all soil parameters and water tables, is perfectly transferred to analytical programs without manual input.

3. Program-to-Program Linking

Calculate earth pressures in one module and seamlessly transfer the resulting forces to another (e.g., from Sheeting Check directly into Spread Footing). The data flows logically and bidirectionally.
